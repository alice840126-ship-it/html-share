코딩 몰라도 10분 만에 완성 — AI로 한국 법령 검색하는 MCP 설치 사용법 (클로드 연동)

공인중개사 실무에서 매일 법령을 찾아야 하는 분, 시험 준비 중인 분, 그리고 AI 도구를 실무에 활용하고 싶은 분을 위한 실전 가이드입니다.

Korean Law MCP가 뭔가요? (비개발자용 1분 요약)


MCP는 Model Context Protocol의 약자로, 쉽게 말하면 Claude(클로드) AI에 외부 도구를 꽂아주는 연결 표준입니다. 스마트폰에 앱을 설치하듯, Claude에 기능을 추가하는 방식이라고 생각하시면 됩니다.


Korean Law MCP는 그 중에서도 국가법령정보센터(법제처)의 Open API를 Claude와 연결해주는 도구입니다. 설치하고 나면 Claude에게 "주택임대차보호법 제3조 알려줘" 또는 "지식산업센터 입주 가능 업종 법령 조항 찾아줘"라고 말하는 것만으로, AI가 법제처 공식 데이터를 실시간으로 가져다 답해줍니다.


이 도구를 만든 분은 광진구청 공무원 출신 개발자 류주무관 님으로, 법제처 API 기반의 89개 도구를 하나의 MCP 패키지에 담아 공개하셨습니다. 법령 검색, 판례, 행정규칙, 조례, 조약까지 망라합니다.


핵심만 요약하면 이렇습니다.

  • 법 조문 번호를 몰라도 됩니다
  • 실시간 최신 법령을 조회합니다
  • Claude Desktop, Claude Code 모두 연동됩니다
  • API 키는 무료로 발급됩니다

공인중개사·부동산 실무자에게 이게 왜 필요한가


저는 부동산에 일하면서, 동시에 2026년 공인중개사 1차 시험을 준비하고 있습니다. Claude Code 환경에서 41개 에이전트를 운영하며 업무 자동화를 꾸준히 이어온 경험이 있는데, Korean Law MCP를 처음 접했을 때 "이건 부동산 실무자를 위해 만든 도구구나"라는 생각이 바로 들었습니다.


부동산 중개 업무는 법령과 떼려야 뗄 수 없습니다. 계약서 한 장을 쓰더라도 민법, 주택임대차보호법, 상가건물 임대차보호법, 공인중개사법이 동시에 적용됩니다. 지식산업센터를 전문으로 하는 경우에는 산업집적활성화법, 제조업 입주 기준, 지원시설 면적 비율 등 훨씬 복잡한 법 구조를 매일 다뤄야 합니다.


기존에는 법제처 사이트를 직접 열어서 검색하거나, 각종 유료 법령 서비스를 이용해야 했습니다. 하지만 Korean Law MCP를 설치하고 나서부터는 Claude와 대화하면서 바로 법령 조문을 확인하고, 계약서 작성 중에도 실시간으로 법적 근거를 체크할 수 있게 되었습니다.


실제로 활용 중인 시나리오를 몇 가지 공유합니다.


매일 아침 법령 변화 자동 브리핑
부동산 관련 법령 개정이 있었는지 매일 아침 Claude에게 확인합니다. 임대차 관련 신규 고시나 시행령 변경 사항을 놓치지 않을 수 있습니다.


지식산업센터 입주 업종 허용 범위 실시간 감지
산업집적활성화법 시행령이 개정되면 입주 가능 업종이 바뀝니다. 매물을 안내하기 전 최신 법령 기준으로 업종 적법성을 확인합니다.


계약 위반 사항 자동 체크
계약서 초안을 붙여넣고 "이 조항이 현행 임대차법과 충돌하는 부분이 있는지 확인해줘"라고 물어보면 Claude가 법령 근거와 함께 검토 결과를 알려줍니다.


공인중개사 시험 법령 조문 즉시 검색
시험 준비 중에 특정 조문의 정확한 문구가 헷갈릴 때, 교재 없이도 Claude에게 물어보면 법제처 원문 그대로 가져옵니다.


부동산 실무자가 아니더라도, 법령을 자주 다루는 모든 분들에게 이 도구는 실질적인 시간 절약 수단이 됩니다.

방법 1: Claude.ai 웹에서 바로 사용 (설치 없음, 가장 쉬움)


설치가 전혀 필요 없는 가장 쉬운 방법입니다. Claude.ai 웹사이트에 접속해서 설정 몇 가지만 추가하면 바로 법령 검색을 시작할 수 있습니다. 단, Claude Pro 또는 Max 요금제($20/월 이상)가 필요합니다.


1단계: 법제처 API 키 무료 발급

  1. open.law.go.kr 접속
  2. 회원가입 후 로그인
  3. 마이페이지 → API 신청 → 인증키 발급
  4. 발급된 영문 인증키 복사 (1~2일 소요)

2단계: Claude.ai 설정에서 MCP 연결 추가

  1. claude.ai 접속 후 로그인
  2. 우측 상단 프로필 → 설정(Settings)
  3. 커넥터(Connectors) 메뉴 클릭
  4. 커스텀 커넥터 추가 버튼 클릭
  5. 아래 URL 입력:
https://korean-law-mcp.fly.dev/mcp?oc=발급받은API키
  1. 저장 후 새 대화 시작

3단계: 바로 질문하기


새 대화창에서 이렇게 물어보시면 됩니다.

"주택임대차보호법 제6조 묵시적 갱신 내용 알려줘"
"공인중개사법에서 중개보수 상한 기준이 뭐야?"
"지식산업센터 입주 가능 업종 관련 법령 찾아줘"


앱 설치, 터미널 명령어 없이 이게 전부입니다.

로컬 설치 방법 (Claude Code/Desktop 기준)


Claude Code 환경에서 사용하거나, 매번 npx로 실행하는 방식 대신 npm 전역 설치를 원하는 분들을 위한 방법입니다. 저는 이 방식으로 사용하고 있습니다.


전제 조건

  • Node.js 18 이상 설치 (터미널에서 node -v로 확인)
  • npm 또는 npx 사용 가능 환경
  • 법제처 API 키 발급 완료

npm 전역 설치


터미널(Mac의 경우 Terminal 앱 또는 iTerm2)을 열고 아래 명령어를 입력합니다.

npm install -g korean-law-mcp

설치가 완료되면 korean-law-mcp 명령어가 전역으로 사용 가능해집니다.


Claude Code settings.json 설정


Claude Code의 settings.json 파일에 아래 항목을 추가합니다.

{
  "mcpServers": {
    "korean-law": {
      "command": "korean-law-mcp",
      "env": {
        "LAW_OC": "여기에_API키_입력"
      }
    }
  }
}

Claude Desktop에서도 동일하게 적용됩니다. command를 npx 방식 대신 직접 실행 방식으로 바꾼 것만 다릅니다.

Claude Code 또는 Claude Desktop을 재시작한 뒤, 아래와 같이 입력해 정상 동작을 확인합니다.

법제처 원문 조문이 그대로 출력된다면 설치 성공입니다.

꼭 알아야 할 주의사항


유용한 도구이지만, 사용 전에 반드시 알아야 할 한계도 있습니다.


법적 조언이 아닙니다
Korean Law MCP는 법제처 공식 데이터를 가져와 보여주는 도구입니다. AI가 법령 내용을 요약하거나 해석할 수는 있지만, 이는 어디까지나 참고용이며 법적 조언이나 변호사 상담을 대체할 수 없습니다. 실제 법적 분쟁이나 계약 문제에서는 반드시 전문가와 상담하시기 바랍니다.


API 키는 개인 정보입니다
법제처 API 키를 settings.json이나 코드에 기록할 때, 해당 파일이 외부에 공개되지 않도록 주의하십시오. GitHub 등에 올리는 경우 API 키가 노출될 수 있습니다.


실시간 법령 기준이지만 시차가 있을 수 있습니다
법제처 API는 최신 법령을 반영하지만, 공포 즉시 업데이트되지 않는 경우도 있습니다. 중요한 사안은 법제처 공식 사이트(law.go.kr)에서 직접 확인하는 것을 권장합니다.


판례 해석에는 한계가 있습니다
법령 조문은 정확히 가져오지만, 판례의 해석이나 적용 범위는 AI가 완벽하게 판단하기 어렵습니다. 판례 검색 결과는 참고 자료로 활용하십시오.


Node.js 버전 확인 필수
npm 설치 시 Node.js 18 미만 환경에서는 오류가 발생할 수 있습니다. 문제가 생기면 node -v로 버전을 먼저 확인하십시오.

자주 묻는 질문 (FAQ)


Q1. 공인중개사 시험 법령을 AI로 바로 검색할 수 있나요?


네, 가능합니다. 공인중개사법, 민법, 주택임대차보호법, 부동산 거래신고법 등 시험에 나오는 법령 조문을 조문 번호만 알면 원문 그대로 가져올 수 있습니다. 조문 번호를 모르더라도 "계약갱신 관련 조항 찾아줘"처럼 키워드로 검색하면 관련 조항을 찾아줍니다. 시험 준비 시 법 조문 암기 확인용으로 매우 유용합니다.


Q2. 주임사법, 상임법, 공인중개사법 등 실무 법령 다 조회되나요?


네, 법제처 국가법령정보센터에 등록된 모든 법령이 조회됩니다. 주택임대차보호법, 상가건물 임대차보호법, 공인중개사법은 물론이고, 산업집적활성화법, 부동산 거래신고 등에 관한 법률, 민법 등 부동산 실무에서 자주 쓰이는 법령이 모두 포함됩니다. 행정규칙, 조례, 시행령, 시행규칙까지 검색 범위가 넓습니다.


Q3. 법 조문 번호 몰라도 쓸 수 있나요?


네, 몰라도 됩니다. "전세보증금 반환 관련 조항"이나 "임대인 수선 의무" 같은 키워드로 검색하면 관련 조문을 찾아줍니다. 물론 조문 번호를 알면 더 정확하고 빠르게 원문을 가져오지만, 모른다고 사용을 못하는 것은 아닙니다. AI와 대화하듯 자연어로 질문하면 됩니다.


Q4. 판례나 법령 해석례도 검색되나요?


판례 검색 도구도 포함되어 있습니다. 다만 판례의 경우 법령 조문처럼 원문 전체를 그대로 가져오는 것보다는, 관련 판례 목록과 요지를 안내하는 방식에 가깝습니다. 판례 전문이 필요한 경우 대법원 판례검색 사이트와 병행해서 활용하시는 것을 권장합니다. 법령 해석례는 법제처 유권해석 데이터를 통해 확인할 수 있습니다.


Q5. 무료로 쓸 수 있나요? 어떻게 시작하나요?


네, 완전 무료입니다. 법제처 API 키는 open.law.go.kr에서 회원가입 후 무료로 발급됩니다. Korean Law MCP 패키지 자체도 오픈소스로 무료입니다. Claude Desktop 무료 플랜에서도 MCP 연동이 가능합니다. 시작 방법은 위의 "설치 없이 5분 만에 시작하는 방법" 섹션을 따라하시면 됩니다.
